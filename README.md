# Plant Leaf Disease Detection and Classification

## Project Overview

This project focuses on detecting and classifying wheat leaf diseases using a hybrid deep learning approach. It helps identify diseases such as rust, blight, and septoria from leaf images.

## Methodology

The system uses a combination of deep learning models (InceptionV3 and Xception) along with a Support Vector Machine (SVM) classifier. It extracts both deep features and color-based features from images to improve classification.

## How It Works

* Input: Wheat leaf image
* Preprocessing and segmentation of the leaf
* Feature extraction using deep learning models
* Classification using SVM

## Output

The system identifies:

* Healthy leaf
* Diseased leaf (specific disease type)
